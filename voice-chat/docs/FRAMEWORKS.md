# Frameworks and References

Last verified date: 2026-07-03
Tested OS: macOS

## Selected Frameworks

- faster-whisper: lightweight Whisper inference wrapper.
  - https://github.com/SYSTRAN/faster-whisper
- Piper TTS: local neural TTS engine.
  - https://github.com/rhasspy/piper
- sounddevice: Python audio I/O over PortAudio.
  - https://python-sounddevice.readthedocs.io/
- py-webrtcvad: VAD wrapper for WebRTC VAD.
  - https://github.com/wiseman/py-webrtcvad
- FastAPI: mock server framework.
  - https://fastapi.tiangolo.com/
- Uvicorn: ASGI server.
  - https://www.uvicorn.org/
- HTTPX: async REST client.
  - https://www.python-httpx.org/
- NumPy: signal/tone generation and array ops.
  - https://numpy.org/
